# Network Sniffing Tools

## Overview

This directory contains network sniffing and analysis tools for educational purposes.

## Tools Included

| Tool | Description | GUI |
|------|-------------|-----|
| wireshark | Protocol analyzer | Yes |
| tcpdump | Packet analyzer | No |
| dsniff | Network auditing | No |
| bettercap | MITM attacks | No |
| ettercap | MITM attack suite | Yes |
| netsniff-ng | High-performance sniffer | No |
| driftnet | Image extractor | Yes |
| scapy | Packet manipulation | No |
| hexinject | Raw packet injection | No |
| tcpflow | TCP flow reconstruction | No |
| bruteshark | Network forensics | Yes |
| above | Real-time monitoring | Yes |
| chaosreader | Session reconstruction | Yes |

## Usage

Each tool has its own directory with:
- Main tool documentation (tool-name.md)
- Usage examples
- Common scenarios
- Troubleshooting guide

## Learning Path

1. Start with `tcpdump` for basics
2. Move to `wireshark` for detailed analysis
3. Learn `scapy` for packet crafting
4. Try `ettercap`/`bettercap` for MITM
5. Use `tcpflow`/`chaosreader` for forensics
